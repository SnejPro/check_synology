## Arguments

| Argument | Description | Possible values | Default | Mandatory |
| ------ | ------ | ------ | ------ | ------ |
| -H | SNMP host | hostname/ip-address | None | YES |
| -v | SNMP version | 1/2c/3 | 3 | NO |
| --port | SNMP port | number | 161 | NO |
| ------ | ------ | ------ | ------ | ------ |
| -u | SNMPv3 - username | string | None | YES (for SNMPv3) |
